Project 3: Visual Identity & Portfolio Mockup

This project involved developing a visual identity and creating a final portfolio page using design principles such as layout balance, hierarchy, and responsiveness.

Tools Used

  • Adobe Illustrator
  • Figma
  • HTML & CSS

Reflection

Throughout this project, I applied multiple design techniques learned during the semester, including layout structure, image scaling, color contrast, and spacing. I began by sketching out a layout and then refined it using Figma. One of my biggest takeaways was learning how to apply visual hierarchy—ensuring that important content draws attention through scale and alignment.

By building this project in HTML and CSS, I also improved my web coding skills, especially with responsive layout and styling techniques. It was rewarding to see the concept go from sketch to polished web-ready design. This experience boosted my confidence in both my design sense and front-end development ability. I now better understand how graphic design and web development go hand in hand.
