Project Overview

Design Fundamentals Website Project

This project explores key themes in both web design and graphic design, generating a range of work that will be presented in your own hand-coded website made from the foundational principles of HTML and CSS.

In semester one you will create a series of 3 projects and one reflection essay relating to graphic design, web design, and typography. Using the fundamentals of HTML and CSS learned in this class you will build a static website to present the graphic, and typographic foundations you have learned.

Using GitHub, you will build your website from scratch and utilize version control to manage your code. You will publish and host your static website using GitHub Pages.

Deadline

Thursday, December 12, 2024 at 13:59

Weighting
